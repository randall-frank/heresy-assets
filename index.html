Heresy: 1499 NT Card Deck

Heresy is a fan expansion of the T.I.M.E Stories board game. In T.I.M.E Stories, you and your friends work together to solve mysteries, puzzles, unravel the plot of enemies who plan to exploit the space-time continuum.

In Heresy, you and your friends will be sent to 1499 Spain. Bob needs you to track down and neutralize a rogue T.I.M.E agent, who is using the Inquisition as to cover up his plans. Will you and your friend be able to find the traitor and stop him before it is too late?

Content on this Site

This site contains card images generated for the Heresy expansion. The imagery is provided in multiple formats:

Raw PNG Files

The raw PNG files are broken up into a series of zip files. Each file contains 25 cards, both front and back faces. In total, there are nearly 150 cards that make up the Heresy deck, so there are 6 zip files one would need to download to access all the card images.

  • Cards 0 - 24
  • Cards 25 - 49
  • Cards 50 - 74
  • Cards 75 - 99
  • Cards 100 - 124
  • Cards 125 - 147

Printable PDF Files

These files include all the images for the deck in Letter or A4 layouts. One can print these files and use cardboard to build cards from the result.

Tabletop Simulator Format

These are a set of 6 images that can be used with the Tabletop Simulator application. The expansion is available in the Tabletop Simulator Steam Workshop.

These files are png images that combine many of the images into a set of tiled images. There are three sets of top and bottom images.

  • Set 1: Top Card Image Bottom Card Image
  • Set 2: Top Card Image Bottom Card Image
  • Set 3: Top Card Image Bottom Card Image

Carlito Font

If you would like to generate the cards yourself, please note that they use the Carlito typeface and expect it to be installed on the system on which the cards are generated.

  • Carlito Regular
  • Carlito Bold
  • Carlito Italic
  • Carlito Bold Italic

How were these files generated?

The source code for the files is included in the github repo heresy-assets.

The build_deck tool was used with the above source files to generate the files. The tool can be installed with 'pip install heresycardbuilder'. The command line is included in the docs in the associated github repo heresy-card-builder.
